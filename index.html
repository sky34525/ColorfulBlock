ColorfulBlock

切换颜色，躲避障碍，越快越强！
一款基于 LI 引擎构建的高速方块生存游戏。

⬇ 前往 GitHub 如何游玩 ↓

🎮 在线 演示预览

点击画面开始 · 方向键移动 · Shift 切换颜色 · 空格加速

🕹 操作方式

⬆⬇
↑ / ↓  or  W / S

控制方块上下移动，穿过障碍之间的空隙

🔀
Shift

切换颜色：在蓝色和黄色之间瞬间变换

⚡
Space

消耗能量加速移动，能量会自动恢复

↩
Enter

游戏结束后重新开始

📋 游戏规则

🟦

蓝色方块

当你是蓝色时碰触蓝色障碍，可以穿越并得分！变为黄色则立即死亡。

🟨

黄色方块

当你是黄色时碰触黄色障碍，可以穿越并得分！变为蓝色则立即死亡。

⬛

黑色方块

无论任何颜色，触碰即死。黑色障碍是最危险的威胁，务必绕行。

🔋

能量系统

绿色能量条显示当前能量。耗尽后加速锁定，恢复至 20% 后才能再次使用。

🔧 技术栈

C++ LI Game Engine OpenGL / GLSL GLM ImGui FreeType OpenSans Font

游戏采用双 Pass 渲染：第一 Pass 将场景渲染至帧缓冲（FBO），
第二 Pass 通过后处理全屏 Quad 实现死亡红屏闪光特效。
